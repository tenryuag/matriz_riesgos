· Metodología y Notas

Cómo se calcularon los números de este reporte.

Alcance del reporte

Este reporte mide únicamente la ventana en la que Claude Code ha sido parte del flujo de trabajo: del 6 de febrero al 18 de mayo de 2026 (~3 meses). Las features y métricas mostradas corresponden a lo entregado en ese periodo, no al ciclo de vida completo del proyecto (que inició en octubre de 2025).

Datos duros

Los conteos de commits (13), archivos modificados (48), líneas agregadas (10,857) y eliminadas (411) provienen directamente del repositorio Git filtrado al periodo del reporte. Son cifras objetivas.

Estimaciones

Las horas "sin Claude" y "con Claude" son estimaciones razonables basadas en: el alcance funcional entregado, benchmarks típicos de desarrollo React + Supabase para módulos de tamaño comparable, y la tarifa promedio reportada en el mercado mexicano para desarrolladores senior con stack equivalente. No representan medición exacta de horas trabajadas.

Tarifas y conversiones

Se usó MXN $500/hr como referencia para dev senior (rango de mercado MX $400–700/hr). Tipo de cambio aproximado USD/MXN: 18. El costo de Claude Code se estimó sobre la suscripción mensual estándar; usuarios con planes empresariales pueden tener tarifas distintas.

Limitación. Este reporte mide el ahorro de horas equivalentes, no el valor estratégico adicional (haber podido lanzar antes, atender a más departamentos, o liberar al equipo para otros proyectos), que típicamente multiplica el retorno calculado aquí.
